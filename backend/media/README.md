# Media Folder

This folder contains media assets for the application.

## Structure

```
media/
├── avatars/          # User profile photos (250x250)
│   └── user_{n}.jpg
├── properties/       # Property listing photos (1200x800)
│   └── prop_{n}_{m}.jpg
├── thumbnails/       # Property thumbnail images (400x300)
│   └── prop_{n}_thumb.jpg
├── documents/        # Verification documents (ID scans, deeds, bills)
│   ├── national_id_{n}.jpg
│   ├── notarized_deed_{n}.pdf
│   └── ...
└── videos/           # Property tour videos (MP4)
    └── prop_{n}_tour.mp4
```

## Notes

- Current files are **placeholders** generated by the seed script
- Replace with real images/documents for production
- Max video size: 50MB
- Recommended image formats: JPEG, WebP
- Document formats: PDF (scans), JPEG (photos of IDs)
